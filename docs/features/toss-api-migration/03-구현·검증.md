**증분 — 국내 장마감 후 현재가 = 정규장 종가 고정 (2026-07-15)**
- 문제: KR `lastPrice`(토스 `/prices`)는 정규장 마감(15:30) 후에도 시간외 체결을 따라 흔들려, 그리드·포트폴리오 현재가가 시간외가로 표시됨(예 000660 시간외 2,070,000 → +8.21%). 토스 앱은 정규장 종가 고정 표시(2,082,000 → +8.83%).
- 신규 `TossChangeCalculator::getKrRegularClose($tossSymbol)` — 'KR·장마감·거래일·오늘 일봉 존재' 4조건 만족 시 오늘 일봉(`/candles?interval=1d`) closePrice 반환, 아니면 null(현행 유지). 캐시 `toss_kr_regclose_{symbol}` 다음 09:00 KST TTL(개장 시 라이브 복귀), '오늘 봉 없음'도 sentinel(0) 캐싱해 매 사이클 중복 호출 차단. 장마감 이후에만 조회 → 캐싱 값은 항상 확정 종가.
- 변경 `TossPriceFetcher::fetchDomestic()`(WS 그리드/관심) · `fetchSingle()`(REST 단건·포트폴리오 경유) KR 경로 — 등락 계산 직전 `getKrRegularClose`가 non-null이면 `lastPrice`를 종가로 치환. 현재가만 종가로 바뀌면 등락(종가 − price-limits 기준가)은 `calculate()`가 자동 정합.
- 정규장 중=라이브 유지 · 휴장/개장 전=현행 유지 · **US 무영향**(프리/애프터 시간외 표시 유지, KR 전용). `TossChangeCalculator` 3분기(기준가) 로직 무변경.

## 개선 — 개장 전·휴장 중 0.00% 초기화 제거(세션 인지형 기준가) (2026-07-15, backend)
- **요청(개발자)**: 장 열리기 전(개장 전·휴장)에는 등락률이 0%로 초기화되지 말고 직전 세션(어제) 하루 등락이 계속 보여야 함(토스 앱 동일).
- **원인**: `TossChangeCalculator::fetchAndCachePrevClose()`가 "오늘 봉 없음"이면 무조건 candles[0](어제 종가)을 기준가로 → 개장 전 lastPrice(=어제 종가)와 같아 0.00%.
- **수정**: 기준가 선택을 3분기 세션 인지형으로.
  1. 오늘 봉 있음 → candles[1](어제 종가) — 현행.
  2. 오늘 봉 없음 + 라이브 세션 → candles[0] — 현행(개장 직후·미국 프리마켓 MU 7/10 보존). 라이브 판정: `MarketSessionService` 재사용 — US=`getUsSession() !== '장마감'`(프리·정규·애프터·주간거래 포함), KR=`getKrSession() === '정규장'`.
  3. 오늘 봉 없음 + 장마감(개장 전·휴장·주말) → candles[1](전전일 종가) → 현재가(어제 종가) 대비 = 어제 하루 등락 유지.
  - **US 프리/애프터는 라이브로 취급**(lastPrice 진행 → candles[0]) — 강제로 candles[1] 로 바꾸면 MU 7/10 부호반전이 재발하므로 '정규장만'이 아니라 '장마감 아님' 기준. 휴장일 판정은 토스 캘린더/Yahoo SPY(`MarketSessionService`)에 위임.
  - **TTL**: 3번 분기 캐시는 **다음 개장 시각까지**만(KR 09:00 KST · US 09:30 ET) — 개장 순간 기준가가 어제 종가로 바뀌어야 stale/부호반전 방지. 1·2번 분기 TTL 무변경(US 16:05 ET · KR 자정).
- **주입**: `TossChangeCalculator` 생성자에 `MarketSessionService` 추가(순환 의존 없음), `AppServiceProvider` 클로저 갱신.
- **변경 파일**: `app/Services/Toss/TossChangeCalculator.php`, `app/Providers/AppServiceProvider.php`, `tests/Unit/TossChangeCalculatorTest.php`.
- **테스트(backend)**: `--filter=TossChangeCalculator` **22 통과**(기존 18 + 신규 4: KR정규장-오늘봉없음·US휴장→전전일·KR장마감 TTL·US장마감 TTL). 기존 KR 장전 테스트는 버그 동작(candles[0])을 검증하던 것이라 수정 동작(candles[1]·어제 등락 유지)으로 갱신. Toss/Quote/PriceService 도메인 144 통과, 회귀 없음.
